Number bases are systems by which quantities can be expressed by a set of numerals with values dependent on their position in the number. Base-10 (AKA decimal) is the number base that we use every day and that we all think in. It uses the digits 0 - 9 and the positions have values in increasing powers of 10.
⠀

Here is an example of how decimal works for the number 4823:
⠀

Position⠀ 3 2 1 0
Digit 4 8 2 3
Value 4000⠀ 800⠀ 20⠀ 3


Each digit has a value equal to its value x 10position.

This can be generalised to base-n by saying that each digit will have the value of the digit’s value x nposition. Base-n will have n unique numerals.
